### Code Snippets

### docker compose
```yaml
db:
   ports:
     - "5432:5432"
```
### pom xml profile node
```xml

<profile>
    <id>jdk-11-config</id>
    <activation>
        <jdk>11</jdk>
    </activation>
    <properties>
        <javadocExecutable>${java.home}/bin/javadoc</javadocExecutable>
    </properties>
</profile>
```
